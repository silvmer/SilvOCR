**Local-First Batch Transcription Station**

SilOCR is a lightweight, privacy-focused tool that allows you to transcribe text from images directly in your browser. Since it runs locally using `Tesseract.js`, your images never leave your device.

## ‚ú® Features
* **Privacy First:** No servers, no uploads. Everything happens on your machine.
* **Batch Processing:** Add multiple images and transcribe them all at once.
* **PWA Ready:** Install it as an app on your phone or desktop.
* **Offline Support:** Works even without an internet connection.
* **Paste Support:** Press `Ctrl+V` to add screenshots directly from your clipboard.

## üöÄ Getting Started
1. Download the repository.
2. Open `index.html` in any modern web browser.
3. (Optional) Choose "Install App" or "Add to Home Screen" to use it as a standalone app.
